TROUBLESHOOTING
---------------
Problem: Can't connect to database
Solution: Check DB_HOST, DB_PORT, DB_USER, DB_PASSWORD in .env

Problem: File upload fails
Solution: Check SUPABASE_URL and SUPABASE_KEY in .env

Problem: Migrations not applied
Solution: Run: python manage.py migrate dashboard

Problem: Tables don't exist
Solution: Run: python manage.py makemigrations && python manage.py migrate
